# 🤖 AI Generator Outfits

AI Generator Outfits is a Flutter-based mobile application that uses AI to transform user images into stylish, futuristic, and modern outfit designs. The app preserves the user’s face, pose, and body structure while generating high-quality, photorealistic outfit variations based on custom prompts.
